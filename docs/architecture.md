# CSLIB FMT Architecture

## Layers
1. Graph layer (basic structures, bounded radius)
2. Logic layer (FO^k syntax, locality radius)
3. Game layer (EF games, duplicator strategies)
4. Type layer (rooted-ball encoding)
5. Invariant layer (cycle space, non-factorization)
6. Example layer (high-girth lifts, separation)

## Principle
Local indistinguishability vs global invariants.
